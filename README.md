# ECOMMERCE-
Headline

Ecommerce Sales Dashboard Using Power BI

Purpose

The purpose of this dashboard is to track and analyze ecommerce sales performance, customer purchasing behavior, profit trends, and payment methods through interactive visualizations. It helps businesses identify profitable products, top customers, and sales growth opportunities for better decision-making and business strategy planning.

Tech Stack
Power BI – Dashboard development and data visualization
Microsoft Excel / CSV – Data collection and storage
DAX (Data Analysis Expressions) – KPI calculations and measures
Power Query – Data transformation and cleaning
Data Source
Ecommerce Sales Dataset
Transactional sales data containing:
Order Amount
Profit
Quantity Sold
Customer Details
Product Categories
Payment Modes
State-wise Sales Information
Features
Interactive ecommerce sales dashboard with filters and slicers
State-wise sales analysis
Monthly profit trend visualization
Customer-wise sales performance tracking
Category and sub-category profit analysis
Payment mode distribution insights
KPI cards for Amount, Profit, Quantity, and AOV
User-friendly and visually attractive dashboard design
